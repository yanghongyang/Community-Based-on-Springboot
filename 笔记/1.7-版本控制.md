使用命令行创建一个新仓库

```bash
echo "# Springboot-projects-a-website-like-nowcoder-" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/yanghongyang/Springboot-projects-a-website-like-nowcoder-.git
git push -u origin main
```

使用命令行 `push` 一个已有仓库

```bash
git remote add origin https://github.com/yanghongyang/Springboot-projects-a-website-like-nowcoder-.git
git branch -M main
git push -u origin main
```
