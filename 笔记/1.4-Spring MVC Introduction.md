### HTTP 

- Hypertext Transfer Protocol
- 用于传输 HTML 等内容的应用层协议
- 规定了浏览器和服务器之间如何通信，以及通信时的数据格式

### Spring MVC

- 三层架构

  - 表现层、业务层、数据访问层

- MVC（一种设计模式，集体解决表现层的问题）

  - Model : 模型层
  - View : 视图层
  - Controller : 控制层

- 核心组件

  - 前端控制器 : DispatcherServlet（调度MVC 的 3 个组件）

  

### Thymeleaf

- 模板引擎
  - 生成动态的HTML

  - 标准表达式、判断与循环、模板的布局
